# S-P-500-Companies-Panel
Data from wikipedia, dataset of S&amp;P500 companies past and present in daily panel form accounting for changes.

Current csv date range, 1976-01-01 to 2025-11-06. To end at later date run the script locally.

Data Source: https://en.wikipedia.org/wiki/List_of_S%26P_500_companies

### Prereqs
1. pandas
2. requests
3. lxml

### Notes
1. Before running the script, replace the email and reason for use in line 4.
2. Read the wikipedia User-Agent policy before (https://foundation.wikimedia.org/wiki/Policy:Wikimedia_Foundation_User-Agent_Policy).
3. If using for academic purposes cite the repository.
